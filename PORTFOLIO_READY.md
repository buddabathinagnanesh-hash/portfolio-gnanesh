### Add a New Component
1. Create `src/components/MyComponent.jsx`
2. Import in `App.jsx`
3. Add to JSX
4. Style with `.reveal` class for animations

### Add a New Hook
1. Create `src/hooks/useMyHook.js`
2. Export function
3. Use in components
4. Component re-renders on state change

### Change Colors
1. Edit `src/index.css` `:root` variables
2. All dependent colors auto-update
3. No other changes needed!

### Add Animations
1. Define `@keyframes` in CSS
2. Apply to elements/classes
3. Use `var(--transition-*)` for timing
4. Add to `.active` state for scroll reveals
